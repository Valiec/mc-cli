# mc-cli
CLI minecraft server console

## Installing

1. Install dependencies (for now, just Python 3, Bash, and the GNU coreutils if you don't have them).
2. Clone or download this repository.
3. `cd` into the repo directory and run install.sh to install the tool. It defaults to installing to `/usr/local/bin`, specify a path as an argument if you want to install somewhere else.

If you have the dependencies, cloning, downloading, and starting the installation script can be done with this one command: `tar -xf <(curl -L https://github.com/Valiec/mc-cli/archive/refs/heads/main.zip) && bash ./mc-cli-main/install.sh`

Once successfully installed using this command (as long as you did not install it as a symlink by adding the `-l` option), it is safe to delete the downloaded copy if you want.
